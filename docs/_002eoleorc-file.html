2.12 The .oleorc file

If there is a file .oleorc located in the home directory, it is read when Neoleo starts up. For example, if you always want to use the a0 reference system, and wish to have the status line on the bottom line, rather than on the second line from the top, your .oleorc file could be 10 

        set-option a0
        set-option status -1

Comments in .oleorc can be added. The line should start with the ‘#’ character.
