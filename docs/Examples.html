2.13.2 Examples

Assume that the spreadsheet contains the figure below, where the block of X’s marks the cursor. 


                 |  C  |  D  |  
                 |     |     |   
              --------------------
                 |     |     |   
              2  |  3  |     |   
              --------------------
                 |     |     |   
              3  | "A" | "B" |   
              --------------------
                 | XXX |     |   
               4 | XXX |  7  |   
              --------------------

Here are some computations in a0 mode. 

                C2 * D4    ⇒     21

                D4 % C2    ⇒     1

                C2 = D4    ⇒     #FALSE

                C3 < D3    ⇒     #TRUE

                C3 & D3    ⇒     "AB"
