1 Quickstart

Most important keys:

C-g

Cancel the current operation

C-q

Quit neoleo. Identical to C-x C-c.

C-s

Save(as) file. Identical to M-x save-spreadsheet.

Other highly useful keys:

C-c

Copy cell formula to clipboard

C-l

Set alignment of current cell left

C-r

Set alignment of current cell right

C-v

Paste formula in current clipboard to current cell

This is by no means a complete list of the key bindings. There are many more described in the rest of this manual.
