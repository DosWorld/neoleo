2 Basics

Neoleo is the GNU spreadsheet 1.

Neoleo has more than one user interface. The traditional neoleo environment shows a curses based (character mode) user interface. A bare bones user interface based on the X Window System exists as of version 1.6 which dates back to 1994. In 1998, development started for a motif based user interface. It should be more user friendly than the character based UI.

To start Neoleo, type neoleo. If filename is the name of a spreadsheet that you have saved from a previous neoleo session, you can type neoleo filename to start Neoleo with the spreadsheet loaded.

To stop Neoleo, type C-x C-c. This will prompt you for the name of a file in which to save the current spreadsheet. If you do not want to save it, type RET, otherwise type the name of the filename that you wish to save it in, and then press RET.

Type C-z to suspend neoleo. This does not do anything if you are running under X-windows.

If you wish to abort a command, type C-g
