6 Multiple Windows

Neolo can split a spreadsheet into two or more windows. The windows showing the spreadsheet can show different parts of it. Only one spreadsheet can be displayed at a time.

• Windows:  Introduction to Neolo windows.
• Split Window:  New windows are made by splitting existing windows.
• Other Window:  Moving to another window or doing something to it.
• Change Window:  Deleting windows and changing their sizes.
